# Resumen del Proyecto: Huerto Sostenible con Compose Multiplatform

## Descripción General
Este es un proyecto de una aplicación móvil desarrollada con **Compose Multiplatform** que se centra en la gestión de un huerto sostenible. La aplicación está diseñada para ayudar a los usuarios a planificar y mantener su huerto, con un enfoque en prácticas ecológicas y sostenibles.

## Características Principales
La aplicación cuenta con una pantalla de inicio (HomeScreen) que actúa como un panel de control, dando acceso a las diferentes secciones:

*   **Inicio de Sesión y Perfil:** La aplicación incluye un sistema de autenticación (probablemente con Google, a juzgar por los archivos del proyecto) que permite a los usuarios tener un perfil (`Mi Perfil`) y guardar su progreso y configuraciones.
*   **Gestión de Bancales:** Es una de las funcionalidades clave. Los usuarios pueden crear y gestionar sus bancales (`Mis Bancales`), donde podrán seleccionar y organizar los cultivos que deseen plantar.
*   **Diario de Cultivo:** Una sección (`Diario de Cultivo`) para que los usuarios puedan llevar un registro detallado de las actividades realizadas en su huerto, como siembras, riegos, cosechas, etc.
*   **Calendario:** La aplicación incluye un calendario (`Calendario`) para ayudar a planificar las tareas agrícolas a lo largo del año.
*   **Información y Ayuda:** Secciones dedicadas a proporcionar información útil al usuario, como una guía de `Plagas` y una sección de `Consejos` para el cultivo.

## Tecnología
*   **UI Framework:** Jetpack Compose para la interfaz de usuario.
*   **Arquitectura:** El código está organizado en `screens` (pantallas), lo que sugiere una arquitectura de UI modular.
*   **Navegación:** Se utiliza un sistema de navegación para moverse entre las diferentes pantallas de la aplicación.
*   **Plataforma:** Compose Multiplatform, lo que indica que la aplicación podría funcionar tanto en Android como en otras plataformas compatibles.

## Estructura del Código (basado en HomeScreen.kt)
La pantalla principal (`HomeScreen`) está construida con un `Scaffold` que contiene una `TopAppBar` y un `LazyVerticalGrid`. La cuadrícula muestra tarjetas (`ElevatedCard`) para cada una de las secciones principales, cada una con un icono y un título, facilitando una navegación visual e intuitiva para el usuario.
